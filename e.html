Troubleshooting Guide

OpenGuessr uses a peer2peer connection system via WebRTC Datachannels. Follow these steps to ensure a smooth experience:

  • Disable VPNs or Proxies: Certain VPNs or Proxies, often browser extensions, block WebRTC connections.

  • Disable Firewalls: Some restrictive firewall settings or wifi routers can block WebRTC.

  • Check if the TURN servers are online: We use turn servers from fastturn.
  • Check if your browser supports WebRTC: Open chrome://webrtc-internals/ or about:webrtc for Firefox.

If multiplayer still does not work after following these steps, consider using a different wifi, cellular data or switch the host.
